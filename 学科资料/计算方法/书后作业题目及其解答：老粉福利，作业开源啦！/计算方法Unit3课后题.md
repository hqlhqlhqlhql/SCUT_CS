### 1.求方程组
`x1 + 2x2 = 1`
`x1 + x2 = 0`
`x1 - x2 = 1`
的最小二乘解。

设系数矩阵`A=[[1,2],[1,1],[1,-1]]`，右端向量`b=[1,0,1]^T`，
最小二乘解满足正规方程组`A^T A x = A^T b`。
`A^T A = [[3,2],[2,6]]`，
`A^T b = [[2],[1]]`，
方程组为：
`3x1 + 2x2 = 2`
`2x1 + 6x2 = 1`
解得：`x1=5/7`，`x2=-1/14`。
