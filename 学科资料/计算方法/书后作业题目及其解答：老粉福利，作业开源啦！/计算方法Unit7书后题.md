## 第七单元：非线性方程与方程组的数值解法

### 第1题

**【题目】** 用对分法求下列方程的根，要求绝对误差限 $\varepsilon = 0.005$。

(1) $x^3 - x + 4 = 0$，区间 $(-2, -1)$；

(2) $2x = e^{-x}$，区间 $(0, 1)$。

**【解】**

**理论前置解释（确定二分次数）**：

根据对分法误差估计公式，经历 $n$ 次二分后，区间半长（即最大可能绝对误差）满足 $\frac{b-a}{2^n} \leq \varepsilon$。

对于本题中两个小题，区间长度 $b-a$ 均为 $1$（即 $-1-(-2)=1$ 和 $1-0=1$）。

$$\frac{1}{2^n} \leq 0.005 \implies 2^n \geq 200 \implies n \geq \frac{\ln 200}{\ln 2} \approx 7.64$$

因此，**两个小题均需要严格对分迭代 8 次**。

**(1) 方程 $f(x) = x^3 - x + 4 = 0$，区间 $[-2, -1]$**

- **第 0 步**：$a_0 = -2, b_0 = -1$。由于 $f(-2) = -2 < 0, f(-1) = 4 > 0$，异号，说明区间内确有实根。
    
- **第 1 次对分**：中点 $c_1 = -1.5$。计算 $f(-1.5) = 2.125 > 0$。替换右边界：$b_1 = -1.5, a_1 = -2$。
    
- **第 2 次对分**：中点 $c_2 = -1.75$。计算 $f(-1.75) = 0.3906 > 0$。替换右边界：$b_2 = -1.75, a_2 = -2$。
    
- **第 3 次对分**：中点 $c_3 = -1.875$。计算 $f(-1.875) = -0.7168 < 0$。替换左边界：$a_3 = -1.875, b_3 = -1.75$。
    
- **第 4 次对分**：中点 $c_4 = -1.8125$。计算 $f(-1.8125) = -0.1396 < 0$。替换左边界：$a_4 = -1.8125, b_4 = -1.75$。
    
- **第 5 次对分**：中点 $c_5 = -1.78125$。计算 $f(-1.78125) = 0.1281 > 0$。替换右边界：$b_5 = -1.78125, a_5 = -1.8125$。
    
- **第 6 次对分**：中点 $c_6 = -1.796875$。计算 $f(-1.796875) = -0.0022 < 0$。替换左边界：$a_6 = -1.796875, b_6 = -1.78125$。
    
- **第 7 次对分**：中点 $c_7 = -1.7890625$。计算 $f(-1.7890625) = 0.0637 > 0$。替换右边界：$b_7 = -1.7890625, a_7 = -1.796875$。
    
- **第 8 次对分**：中点 $c_8 = -1.79296875$。
    
    此时区间长度已缩减至 $0.0078125$，当前中点离真实根的绝对误差绝对小于 $\frac{0.0078125}{2} = 0.0039 < 0.005$。
    
- **结论**：取三位小数近似解为 **$x \approx -1.793$**。
    

**(2) 方程 $f(x) = 2x - e^{-x} = 0$，区间 $[0, 1]$**

- **第 0 步**：$a_0 = 0, b_0 = 1$。$f(0) = -1 < 0, f(1) \approx 1.6321 > 0$，满足零点存在定理。
    
- 进行 8 次对分逻辑迭代（数据列表见下）：
    
    - $c_1 = 0.5 \implies f(0.5) > 0 \implies b_1 = 0.5$
        
    - $c_2 = 0.25 \implies f(0.25) < 0 \implies a_2 = 0.25$
        
    - $c_3 = 0.375 \implies f(0.375) > 0 \implies b_3 = 0.375$
        
    - $c_4 = 0.3125 \implies f(0.3125) < 0 \implies a_4 = 0.3125$
        
    - $c_5 = 0.34375 \implies f(0.34375) < 0 \implies a_5 = 0.34375$
        
    - $c_6 = 0.359375 \implies f(0.359375) > 0 \implies b_6 = 0.359375$
        
    - $c_7 = 0.3515625 \implies f(0.3515625) < 0 \implies a_7 = 0.3515625$
        
    - $c_8 = 0.35546875 \implies f(0.35546875) > 0 \implies b_8 = 0.35546875$
        
- **最终求中点**：最后锁定区间为 $[0.3515625, 0.35546875]$，中点为 $0.353515625$。
    
- **结论**：满足误差要求，解为 **$x \approx 0.3535$**。
    

### 第4题

**【题目】** 证明方程 $f(x) = 5x^2 - 120x + 30 = 0$ 在区间 $[0,1]$ 中有唯一实根。并验证下面两种不动点迭代格式的收敛性：

(1) $x = \varphi_1(x) = 5x^2 - 119x + 3$；

(2) $x = \varphi_2(x) = \frac{5x^3 + 3}{120}$。

**【证明/解】**

**第一部分：唯一实根性证明**

1. **存在性**：计算连续函数 $f(x)$ 两个端点值：$f(0) = 30 > 0$ ， $f(1) = 5 - 120 + 30 = -85 < 0$。由于 $f(0) \cdot f(1) < 0$，根据介值定理（零点定理），区间 $[0,1]$ 内必至少存在一个实根。
    
2. **唯一性**：求导数 $f'(x) = 10x - 120$。在区间 $x \in [0, 1]$ 上，恒有 $f'(x) \leq -110 < 0$。单调递减性质说明函数曲线与 $x$ 轴只会相交一次，故**区间内存在唯一实根**。
    

**第二部分：验证迭代格式收敛性（带原理解释）**

- **解释**：不动点迭代格式 $x_{k+1} = \varphi(x)$ 局部收敛的充要/充分条件是：在方程的精确根 $x^*$ 附近（或整个考察区间内），其导数的绝对值严格小于 1，即 $|\varphi'(x)| < 1$。
    
- **(1) 格式一**：$\varphi_1(x) = 5x^2 - 119x + 3$
    
    求导得：$\varphi_1'(x) = 10x - 119$。当 $x \in [0,1]$ 时， $|\varphi_1'(x)| \in [109, 119]$。
    
    因为 $|\varphi_1'(x)| \gg 1$，这表明在根附近任何微小的偏差都会被放大一百多倍，导致计算剧烈震荡，所以该格式**发散**。
    
- **(2) 格式二**：$\varphi_2(x) = \frac{5x^3 + 3}{120}$
    
    求导得：$\varphi_2'(x) = \frac{15x^2}{120} = \frac{x^2}{8}$。
    
    当 $x \in [0,1]$ 时，恒有 $0 \leq \varphi_2'(x) \leq \frac{1}{8} = 0.125 < 1$。
    
    同时，当 $x \in [0, 1]$ 时，$\varphi_2(x)$ 的取值范围是 $[0.025, 0.0667] \subset [0, 1]$（满足自映射）。根据压缩映射原理，此格式**必定收敛**。
    

### 第9题

**【题目】** 写出牛顿（Newton）迭代法格式，并用牛顿法计算：

(1) 不开平方计算 $\sqrt{10}$，精确到5位小数；

(2) 方程 $x^3 - x - 1 = 0$ 在 $1.3$ 附近的根，精确到5位小数。

**【解】**

**通用牛顿迭代法格式**：

对于非线性方程 $f(x) = 0$，其切线递推（Newton-Raphson）格式为：

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

**(1) 计算 $\sqrt{10}$**

- **构造方程**：令 $x = \sqrt{10} \implies f(x) = x^2 - 10 = 0$。导数 $f'(x) = 2x$。
    
- **牛顿格式**：$x_{n+1} = x_n - \frac{x_n^2 - 10}{2x_n} = \frac{1}{2}\left(x_n + \frac{10}{x_n}\right)$。
    
- **精细迭代过程**（选初值 $x_0 = 3$）：
    
    - $x_1 = \frac{1}{2}\left(3 + \frac{10}{3}\right) = \frac{19}{6} \approx 3.16667$
        
    - $x_2 = \frac{1}{2}\left(\frac{19}{6} + \frac{10}{19/6}\right) = \frac{721}{228} \approx 3.1622807$
        
    - $x_3 = \frac{1}{2}\left(3.1622807 + \frac{10}{3.1622807}\right) \approx 3.1622777$
        
- **结论**：由于 $x_2$ 和 $x_3$ 在小数点后五位已完全重合，停止迭代。结果为 **$\sqrt{10} \approx 3.16228$**。
    

**(2) 求方程 $f(x) = x^3 - x - 1 = 0$ 在 $1.3$ 附近的根**

- **准备项**：$f'(x) = 3x^2 - 1$。
    
- **牛顿格式**：$x_{n+1} = x_n - \frac{x_n^3 - x_n - 1}{3x_n^2 - 1}$。
    
- **精细数值计算过程**（初值 $x_0 = 1.3$）：
    
    - $n=0$：$f(1.3) = -0.103$， $f'(1.3) = 4.07$。
        
        $x_1 = 1.3 - \frac{-0.103}{4.07} \approx 1.3 + 0.0253071 = 1.3253071$
        
    - $n=1$：$f(1.3253071) \approx 0.0031952$， $f'(1.3253071) \approx 4.2692040$。
        
        $x_2 = 1.3253071 - \frac{0.0031952}{4.2692040} \approx 1.3253071 - 0.0007484 = 1.3245587$
        
    - $n=2$：$f(1.3245587) \approx 0.0000030$， $f'(1.3245587) \approx 4.2633215$。
        
        $x_3 = 1.3245587 - \frac{0.0000030}{4.2633215} \approx 1.3245587 - 0.0000007 = 1.3245580$
        
- **结论**：数值稳定至五位小数，最终解为 **$x \approx 1.32456$**。
    

### 第11题

**【题目】** 给定方程 $f(x) = 0$，构造迭代格式 $x_{k+1} = x_k + c f(x_k)$，其中 $c$ 为非零常数。若 $a$ 为方程的精确根（即 $f(a) = 0$ 且 $f'(a) \neq 0$），试求使得该迭代法在根 $a$ 附近具有局部收敛性的 $c$ 的取值范围。

**【解】**

该格式的迭代函数为 $\varphi(x) = x + c f(x)$。

根据非线性方程单点迭代的**局部收敛性定理**，若想让迭代格式在不动点 $a$ 附近收敛，其一阶导数在不动点处必须满足绝对值严格小于 1：

$$|\varphi'(a)| < 1$$

1. 首先对迭代函数求导数：
    
    $$\varphi'(x) = 1 + c f'(x)$$
    
2. 代入精确根 $x = a$：
    
    $$\varphi'(a) = 1 + c f'(a)$$
    
3. 建立不等式求解 $c$：
    
    $$-1 < 1 + c f'(a) < 1$$
    
    各部分同时减去 1 得：
    
    $$-2 < c f'(a) < 0$$
    
4. 由于题目已知 $f'(a) \neq 0$，我们必须根据导数的正负号分情况讨论（保证除以负数时不等式方向转变）：
    
    - **情况一**：若 $f'(a) > 0$，同除以正数，不等式方向不变：
        
        $$-\frac{2}{f'(a)} < c < 0$$
        
    - **情况二**：若 $f'(a) < 0$，同除以负数，不等式方向颠倒：
        
        $$0 < c < -\frac{2}{f'(a)}$$
        

**【大白话总结答案】**

常数 $c$ 的选取符号必须与 $f'(a)$ **相反**（异号），且其绝对值不能太大，范围控制在：

$$|c| < \frac{2}{|f'(a)|}$$
